First Law of Thermodynamics

Thermodynamics

The First Law of Thermodynamics, also known as the Law of Energy Conservation, states that energy cannot be created or destroyed in an isolated system. It is a fundamental concept describing how energy behaves during chemical and physical changes.

Key Formula

Mathematically, the First Law of Thermodynamics can be expressed as:

ΔU = q + w

where:

  • ΔU = Change in internal energy of the system
  • q = Heat added to the system
  • w = Work done on the system

Understanding the First Law

In practical terms, the First Law tells us that the energy of a system is affected by the heat added or removed from it and the work done on or by it. This energy is stored as internal energy within the system.

Types of Processes

The First Law applies differently in various thermodynamic processes:

  • Isothermal Process: Temperature remains constant (ΔU = 0, so q = -w).
  • Adiabatic Process: No heat is exchanged with surroundings (q = 0, so ΔU = w).
  • Isochoric Process: Volume remains constant (w = 0, so ΔU = q).
  • Isobaric Process: Pressure remains constant (ΔU = q - PΔV).

Application Example

Consider a gas inside a piston. When heat is added to the gas, it expands, pushing against the piston, which does work on the surroundings. According to the First Law, the energy change in the gas (internal energy) equals the heat added plus the work done by the gas.

Implications of the First Law

This law is fundamental to understanding energy transfer in reactions, engine cycles, and various processes in chemistry and physics. It also sets the foundation for the concepts of enthalpy, entropy, and the later laws of thermodynamics.

                    cin >> processType;
                
                    if (processType == 1) {
                        cout << "Select the type of reversible process:\n";

                        if (reversibleType == 1) {
                            double n, T, Vi, Vf, R = 8.314;
                            cout << "Enter the number of moles (n): ";
                            cin >> n;
                            cout << "Enter the temperature (T in K): ";
                            cin >> T;
                            cout << "Enter the initial volume (Vi in m^3): ";
                            cin >> Vi;
                            cout << "Enter the final volume (Vf in m^3): ";
                            cin >> Vf;
                            W = n * R * T * log(Vf / Vi);
                            cout << "The work done (W) in an isothermal process is: " << W << " J" << endl;
                
                        } else if (reversibleType == 2) {
                            double P, deltaV;
                            cout << "Enter the pressure (P in Pa): ";
                            cin >> P;
                            cout << "Enter the change in volume (ΔV in m^3): ";
                            cin >> deltaV;
                            W = P * deltaV;
                            cout << "The work done (W) in an isobaric process is: " << W << " J" << endl;
                
                        } else if (reversibleType == 3) {
                            W = 0;
                            cout << "The work done (W) in an isochoric process is: " << W << " J (no volume change)" << endl;
                
                        } else {
                            cout << "Invalid choice." << endl;
                        }
                
                    } else if (processType == 2) {
                        double Q, deltaU;
                        cout << "Enter the heat added to the system (Q): ";
                        cin >> Q;
                        cout << "Enter the change in internal energy (ΔU): ";
                        cin >> deltaU;
                        W = Q - deltaU;
                        cout << "The work done (W) in an irreversible process is: " << W << " J" << endl;
                
                    } else {
                        cout << "Invalid choice." << endl;
                    }
                
                    return 0;
                }

            #include < iostream >

                int main() {
                    double Q, W, U;
                    
                    cout << "Enter the heat added to the system (Q): ";
                    cin >> Q;
                    
                    cout << "Enter the work done on the system (W): ";
                    cin >> W;
